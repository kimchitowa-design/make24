# 🎯 Make 24 ゲーム

4つの数字を使って24を作る計算パズルゲーム！

## 🎮 遊び方

1. `index.html` をブラウザで開く
2. 表示された4つの数字を計算機ボタンで入力
3. 答えが24になればクリア！

## 📲 アプリとしてインストール (PWA)

このゲームはPWA (Progressive Web App) に対応しています。
PCやスマートフォンのホーム画面に追加して、アプリのように遊ぶことができます。

- **PC (Chrome/Edge)**: アドレスバーの右側にある「インストール」アイコンをクリック
- **Android**: 「ホーム画面に追加」を選択
- **iOS**: Safariの「共有」ボタンから「ホーム画面に追加」を選択

## ✨ 機能

- **3つのレベル**: 
  - **ふつう（18問）**: 
    - 括弧 **なし** で解ける問題
    - 演算子: `+`, `-`, `×`, `÷`
  - **むずかしい（6問）**: 
    - **×（掛け算）** と **括弧** を使う問題
    - 演算子: `+`, `-`, `×`, `÷`, `(`, `)`
  - **鬼（6問）**: 
    - **括弧** と **÷（割り算）** を使う最難関問題
    - 演算子: `+`, `-`, `×`, `÷`, `(`, `)`
- **計算機ボタン**: ボタンで数字と演算子を入力（キーボード入力は無効）
- **問題ナビゲーション**: 前の問題・次の問題ボタンで自由に移動
- **回答履歴保持**: 一度回答した問題は採点まで再挑戦不可、結果を保持
- **解答一例表示**: 困ったときは解答一例ボタンで確認（正解率に影響）
  - 回答済みの問題でも解答一例を確認可能
- **採点機能**: 現在のレベルの正解数と正解率を確認
  - 正解数に応じた名言メッセージを表示
  - レベルごとに異なるメッセージ
- **統計情報**: 正解率と連続正解数を表示
- **レベル別進行状況**: 各レベルの問題番号を個別に保持
- **美しいUI**: モダンでアニメーション付きのデザイン
- **レスポンシブ対応**: PC・スマホどちらでも快適にプレイ可能

## 📐 ルール

- 4つの数字すべてを1回ずつ使う
- 指定された演算子と括弧を駆使する
- 答えが24になればクリア！
- 一度回答した問題は採点するまで再挑戦不可
- 解答一例を表示した問題は不正解扱い

## 💡 例題

**レベル1（ふつう）**  
数字: 6, 6, 6, 6  
解答例: `6 + 6 + 6 + 6`

**レベル2（むずかしい）**  
数字: 1, 2, 3, 4  
解答例: `(1 + 2 + 3) * 4`

**レベル3（鬼）**  
数字: 8, 8, 3, 3  
解答例: `8 / (3 - 8/3)`

## 🎯 操作方法

- **数字ボタン**: 問題の4つの数字を入力（各1回のみ）
- **演算子ボタン**: +、-、×、÷、(、) を入力
- **BackSpace**: 最後の文字を削除
- **クリア**: 入力をすべて削除
- **= ボタン**: 答えを確認
- **⬅️ 前の問題**: 前の問題に戻る
- **➡️ 次の問題**: 次の問題に進む
- **📝 解答一例**: 解答一例を表示（正解率に影響）
- **📊 採点する**: 現在のレベルの採点結果を表示

## 🏆 採点メッセージ

各レベルで正解数に応じた名言メッセージが表示されます！

**ふつうレベル**: 正解率に応じた励ましメッセージ  
**むずかしいレベル**: 努力や成長に関する名言  
**鬼レベル**: 熱い戦いの名言
